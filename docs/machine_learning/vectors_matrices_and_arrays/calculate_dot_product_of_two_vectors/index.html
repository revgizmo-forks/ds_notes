Calculate Dot Product Of Two Vectors

December 20, 2017

Preliminaries

# Load library
import numpy as np

Create Two Vectors

# Create two vectors
vector_a = np.array([1,2,3])
vector_b = np.array([4,5,6])

Calculate Dot Product (Method 1)

# Calculate dot product
np.dot(vector_a, vector_b)
32

Calculate Dot Product (Method 2)

# Calculate dot product
vector_a @ vector_b
32
