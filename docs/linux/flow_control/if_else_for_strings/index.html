If Else For Strings

June 17, 2018

While you can often use [[ ]] for integers, (( )) is specifically designed to work with integers and provides a lot more functionality.

Create Variable

name="Ralph Holmes"

Create If Else Statement

if [[ "$name" == "Ralph Holmes" ]]; then
    echo "It is Ralph."
else
    echo "It is not Ralph"
fi
It is Ralph.
