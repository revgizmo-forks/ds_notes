Create Bucket

Make a bucket (mb) on AWS S3 called kicks-pasta-steer. The bucket name you choose must be globally unique, meaning nobody else in the world must have used that bucket name before.

aws s3 mb s3://kicks-pasta-steer
make_bucket: kicks-pasta-steer
